O computador


Uma dúvida que atinge a maioria dos usuários de PC. Quais são os componentes principais do computador? Para que eles servem?




Um computador é formado por componentes importantes para seu correto funcionamento. Sem um deles, o PC pode até não ligar. São eles:


Dispositivo de Armazenamento

Conhecidos como HD ou disco rígido, é o responsável por armazenar todos os dados do computador. É onde fica instalado o sistema operacional, programas, jogos e outros arquivos (música, vídeos e fotos). Até o momento existe mais dois tipos de armazenamento: o SSHD (chamado também de Híbrido) e o SSD.

Processador

Conhecido também como CPU (Central Processing Unit – Unidade Central de Processamento), é o responsável pela leitura de todos os dados do computador. Tudo o que está acontecendo em sua tela é o processador quem está lendo os dados. Desde a inicialização do sistema operacional até a execução de um programa é o processador quem está fazendo.

Memória RAM

A memória RAM (Random Access Memory – Memória de acesso aleatório) serve para armazenar dados de programas enquanto eles são executados. É a responsável por fazer uma “ponte” entre o dispositivo de armazenamento (HD, SSHD ou SSD) e o processador. Sem a memória Ram, o processador teria o trabalho extra de buscar os dados no armazenamento e ter que os ler ao mesmo tempo.

Placa-mãe

É a responsável por interligar todos os componentes principais do computador. É nela que está o processador, a memória Ram e a ligação com o armazenamento. Além de interligar todos esses componentes, é nela que conectamos o teclado, mouse, monitor, pen drive, cabo da internet e saída de som.

Fonte

Para todos os componentes funcionarem, precisa de energia. A fonte alimenta a placa-mãe, o dispositivo de armazenamento e outros componentes (que não são tão uteis para alguns usuários).

Gabinete

É o esqueleto, a carcaça que irá manter todas as peças fixas. Prendemos em um gabinete a placa-mãe, o dispositivo de armazenamento, a fonte, etc.

Leitor de mídia física

Chamado também de driver de CD/DVD/Bluray, é o responsável por ler os dados armazenados em mídias físicas. Hoje com o avanço da mídia digital, os discos não estão sendo mais tão comercializados (como é o caso de CDs de música).

Estabilizador

Um dos componentes mais controversos que existe em um PC. Alguns usuários utilizam e outros não. Em geral ele serve para alimentar a energia do computador, equilibrando a voltagem que vem da tomada (utilizado para prevenir oscilação de energia na rede elétrica).

Placa de vídeo 3D

Hoje em dia chamada também de PCI Express, placa VGA ou GPU (Graphics Processing Unit – Unidade de Processamento Gráfico), é a responsável pela renderização de imagens em 3D. Um Computador pode funcionar sem ela para visualizar fotos, vídeos, navegar pela internet e rodar alguns programas. A Placa-mãe já vem com um chip gráfico para executar tarefas mais simples, mas para jogos e programas mais pesados é obrigatório ter uma placa de vídeo 3D.

Nobreak

Um acessório importante para qualquer computador (empresas principalmente). Ele funciona como uma bateria gigante que alimenta o computador quando tem quedas de energia. Ele ajuda a evitar falhas técnicas dos componentes principais do PC (HD, processador, placa-mãe) devido a variação de energia na rede elétrica. Muitos usuários não o utilizam pelo seu preço e por achar que o estabilizador se encarrega de “segurar a energia”.
